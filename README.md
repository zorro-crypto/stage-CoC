# CoC Investigator Keeper

A small Chub Stage for running Call of Cthulhu sessions.

## Features

- Register multiple investigators.
- Manually increase or decrease HP, MP, SAN, and Luck.
- Toggle major wound status.
- Keep notes for temporary insanity, indefinite insanity, inventory, and clues.

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The main stage implementation is in `src/Stage.tsx`.
